<h1>Discord Mafia Bot</h1>

<h2>A Bot for Helping Hosts Run Discord Mafia Games</h2>

<p>main.js contains core structures for running the bot, including an onMessage command handler.</p>

<p>The commands directory contains the commands used by the program. You can add additional functions to the bot by adding additional command files here.</p>

------------------------------------------------------------------------------------

Things you will need:

<p>For instructions on setting up a bot to use these files, see this video:
https://www.youtube.com/watch?v=j_sD9udZnCk</p>

<p>You will need a token.json file to store your token in as an export. It should look something like:

{
    "token":"your bot's token here"
}</p>
